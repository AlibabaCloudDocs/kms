# 概述 {#concept_69005_zh .concept}

对 KMS API 接口调用是通过向 KMS API 的服务端地址发送 HTTP POST 和 GET 请求，并按照接口说明在请求中加入相应请求参数来完成的；根据请求的处理情况，系统会返回处理结果。
